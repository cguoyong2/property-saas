# 07-PC管理端页面清单

## 1. 平台运营端页面

| 页面 | 路由建议 | 核心功能 | 接口 |
|---|---|---|---|
| 平台工作台 | `/platform/dashboard` | 租户数、到期数、项目数、用量、接口异常 | `/api/platform/dashboard` |
| 租户管理 | `/platform/tenants` | 新增、编辑、启用、停用、续费、归档 | `/api/platform/tenants` |
| 租户套餐 | `/platform/packages` | 套餐、模块、额度、价格、周期 | `/api/platform/packages` |
| 租户配置 | `/platform/tenants/:tenantId/config` | Logo、域名、支付、小程序、短信、存储 | `/api/platform/tenants/{tenantId}/config` |
| 用量统计 | `/platform/usage` | 项目数、房屋数、账号数、存储、短信、接口 | `/api/platform/usage` |
| 平台公告 | `/platform/notices` | 给租户管理员发布公告 | `/api/platform/notices` |
| 运维监控 | `/platform/monitor` | 接口异常、支付回调、短信失败、慢查询 | `/api/platform/monitor` |
| 任务中心 | `/jobs` | 手动触发 SLA、巡检漏检、合同到期、消息派发任务 | `/api/jobs/*` |
| 操作审计 | `/platform/audit` | 查询平台和租户高风险操作 | `/api/system/operation-logs` |

## 2. 租户管理端页面

| 页面 | 路由建议 | 核心功能 |
|---|---|---|
| 工作台 | `/dashboard` | 经营概览、待办、收费概览、工单概览 |
| 项目/小区管理 | `/base/projects` | 项目新增、编辑、状态、负责人、收款主体 |
| 楼栋管理 | `/base/buildings` | 楼栋、区域、商铺区、停车场区域 |
| 单元管理 | `/base/units` | 单元维护 |
| 房屋管理 | `/base/houses` | 房屋导入、编辑、面积、用途、状态 |
| 业主/住户管理 | `/base/members` | 业主、租户、家属、绑定审核 |
| 车辆管理 | `/base/vehicles` | 车牌、车位、月租状态、人员关联 |
| 收费项目 | `/fee/items` | 物业费、停车费、水电费、押金等 |
| 收费标准 | `/fee/standards` | 按面积、户、车位、合同、固定金额、公式 |
| 账单管理 | `/fee/bills` | 生成、导入、催缴、作废、导出 |
| 收款订单 | `/payment/orders` | 订单、支付状态、核销关系 |
| 支付流水 | `/payment/transactions` | 第三方交易号、渠道、金额、对账 |
| 退款管理 | `/payment/refunds` | 申请、审批、提交、回调、状态 |
| 公告通知 | `/service/notices` | 发布、定向、阅读统计 |
| 工单管理 | `/service/workorders` | 受理、派单、处理、回访、评价 |
| 投诉建议 | `/service/complaints` | 投诉、建议、处理、回访 |
| 消息记录 | `/service/messages` | 站内信发送记录、发送状态、失败原因 |
| 消息模板 | `/service/message-templates` | 模板编码、渠道、标题和内容模板 |
| 巡检资产 | `/patrol/assets` | 设备、资产、二维码/NFC |
| 巡检点位 | `/patrol/points` | 点位、位置、要求 |
| 巡检计划 | `/patrol/plans` | 频率、人员、自动任务 |
| 巡检任务 | `/patrol/tasks` | 待办、异常、整改 |
| 可租资源 | `/lease/resources` | 房屋、商铺、车位、广告位等资源 |
| 意向客户 | `/lease/customers` | 来源、跟进、看房、需求 |
| 租赁合同 | `/lease/contracts` | 合同、附件、租金、到期提醒 |
| 门禁设备 | `/device/access` | 设备、人员权限、同步日志 |
| 访客管理 | `/device/visitors` | 访客邀请、二维码、有效期 |
| 停车对接 | `/device/parking` | 车辆同步、停车记录、接口日志 |
| 报表中心 | `/reports` | 收费、工单、巡检、租赁、经营报表 |
| 导入批次 | `/import/batches` | 导入批次、状态、成功失败数、错误报告 |
| 文件对象 | `/files` | 文件元数据、模块、大小、上传方、下载入口 |
| 系统设置 | `/system` | 用户、角色、菜单、部门、字典、模板 |

## 3. 页面验收通用要求

每个页面至少包含：

- 列表筛选；
- 分页；
- 新增/编辑/查看；
- 权限控制；
- 空状态；
- 加载状态；
- 错误提示；
- 导入/导出权限控制；
- 操作日志可追溯。
