# 05-权限矩阵

## 1. 角色分层

| 层级 | 典型角色 | 说明 |
|---|---|---|
| 平台级 | 平台超级管理员、平台运营、平台客服、平台运维、平台财务 | 管理租户、套餐、平台参数、平台日志和用量 |
| 租户总部级 | 物业公司管理员、总部财务、总部客服、总部工程主管、总部招商主管 | 管理本租户全部项目和公共配置 |
| 项目级 | 项目经理、项目财务、客服管家、工程人员、巡检人员、招商人员 | 管理授权项目内数据 |
| 用户级 | 业主、家属、住户、租户、访客 | 使用微信小程序服务 |

## 2. 权限类型

| 权限类型 | 控制内容 |
|---|---|
| 菜单权限 | 用户可见哪些菜单 |
| 按钮权限 | 新增、编辑、删除、审核、导入、导出、退款、作废等动作 |
| 接口权限 | 后端接口是否可调用 |
| 数据权限 | 公司级、项目级、楼栋级、本人、本部门 |
| 套餐权限 | 租户是否购买/启用某模块 |
| 高风险权限 | 账单作废、退款、批量导出、导入覆盖、接口配置等 |

## 3. 模块权限矩阵

| 模块 | 平台运营方 | 物业总部 | 项目人员 | 业主/住户端 |
|---|---|---|---|---|
| 租户管理 | 开通、停用、续费、套餐、用量、日志 | 查看本租户状态 | 无 | 无 |
| 基础档案 | 协助导入，默认不维护业务数据 | 全租户项目、组织、房屋、员工管理 | 授权项目内维护/查看 | 查看个人房屋 |
| 收费账务 | 不操作租户账款 | 模板、账单、订单、退款、报表 | 项目内催缴、线下收款 | 在线缴费、缴费记录 |
| 工单服务 | 只看运行异常 | 工单类型、SLA、报表 | 受理、派单、处理、回访 | 提交、查看、评价 |
| 巡检资产 | 无 | 计划、资产、统计 | 执行、异常上报 | 无 |
| 招商租赁 | 无 | 资源、客户、合同、租金 | 跟进、看房、合同协助 | 查看本人租赁信息 |
| 门禁停车 | 通道配置 | 接口参数、同步规则 | 权限审核、记录查看 | 访客、开门、车辆 |
| 报表 | 平台租户用量 | 本租户经营报表 | 授权项目报表 | 个人记录 |

## 4. 高风险权限清单

| 权限 | 风险 | 控制要求 |
|---|---|---|
| `fee:bill:updateAmount` | 修改应收金额 | 二次确认、原因、日志、审批可配置 |
| `fee:bill:void` | 作废账单 | 禁止作废已核销账单，必须记录原因 |
| `payment:refund:create` | 资金退款 | 权限、审批、原支付校验、回调幂等 |
| `base:member:export` | 个人信息泄露 | 脱敏、导出日志、审批可配置 |
| `system:tenant:suspend` | 影响租户使用 | 仅平台超管，必须确认 |
| `device:config:update` | 影响设备联动 | 运维权限，修改前后留痕 |
| `data:import:batch` | 批量污染数据 | 预校验、错误报告、批次号 |
