# TASK-042 生产上线资料与预生产演练准备包

## 任务目标

在不接入真实微信支付、不接入第三方厂商、不改业务代码的前提下，补齐正式部署前可执行的生产上线资料责任清单和预生产演练执行清单，并刷新交付索引与最新完成包口径。

## 执行范围

1. 新增按责任角色聚合的生产上线资料责任清单。
2. 新增预生产演练执行清单，覆盖解压、环境变量、迁移、构建、冒烟、回滚和 Go/No-Go。
3. 更新最终交付索引、最终验收记录、README、发行说明、任务清单和 MANIFEST。
4. 生成 TASK-042 完成包并复核包体。

## 验收标准

- 上线资料责任清单覆盖项目负责人、运维、DBA、后端、前端、小程序、支付、测试和厂商负责人。
- 预生产演练清单覆盖从最新完成包解压到 Go/No-Go 签核的完整步骤。
- 文档不记录明文密码、私钥、证书或 API key。
- 最新交付索引指向 TASK-042 完成包。
- 完成包不包含 `node_modules`、`dist`、`target`、`unpackage`、`data/uploads`。

## 未完成事项

真实微信支付、生产资料实际填写、第三方厂商对接、真实对象存储、短信/微信模板消息和云监控告警仍需在正式部署或设备安装阶段由项目方提供资料后执行。
