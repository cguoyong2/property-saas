# TASK-044 PC 与小程序页面缺口补齐

## 任务目标

根据技术开发文档和页面清单，补齐当前不依赖真实生产资料、真实微信支付商户资料、真实第三方设备安装联调即可开发的 PC 管理端和微信小程序页面缺口。

## 执行范围

1. PC 管理端补齐租户配置、任务中心、投诉建议、消息记录、消息模板、导入批次、文件对象页面入口。
2. 小程序补齐账单详情、工单详情、工单评价、投诉建议独立提交和我的房屋解绑能力。
3. 后端补齐小程序会员自助解绑接口，并补齐投诉建议聚合查询接口。
4. 补齐小程序房屋解绑权限迁移。
5. 运行后端测试、PC 构建、小程序 H5 与微信小程序构建。

## 验收标准

- `mvn test` 通过。
- `npm run build` 在 `admin-web` 通过。
- `npm run build:h5` 在 `mobile-uniapp` 通过。
- `npm run build:mp-weixin` 在 `mobile-uniapp` 通过。
- 完成包不包含 `node_modules`、`dist`、`target`、`unpackage`、`data/uploads`。

## 后置事项

微信支付真实拉起、生产资料真实值、MinIO/S3 真实资料和第三方设备真实厂商联调继续按项目方确认后置到正式部署或设备安装阶段。
