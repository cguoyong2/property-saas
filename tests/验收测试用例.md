# 验收测试用例

## 1. SaaS 与租户隔离

| 编号 | 场景 | 操作 | 预期结果 |
|---|---|---|---|
| T001 | 创建两个租户 | 平台端创建物业A、物业B | 两个租户状态独立，管理员账号独立 |
| T002 | 租户数据隔离 | A租户创建房屋，B租户查询房屋 | B租户不可见A租户房屋 |
| T003 | 文件隔离 | A租户上传业主证明，B租户尝试访问URL | 返回无权限或文件不存在 |
| T004 | 缓存隔离 | A/B租户相同用户ID菜单缓存 | 菜单不串租户 |
| T005 | 报表隔离 | 平台/租户分别查看报表 | 租户端只看到本租户数据 |
| T006 | 租户停用 | 平台停用租户A | 租户A按规则限制登录和接口调用 |

| F006 | 重复回调 | 同一回调发送两次 | 第二次幂等处理，不重复入账 |
| F007 | 退款 | 对已支付账单申请退款 | 生成退款单和退款流水，账单状态正确 |
| F008 | 作废 | 作废未支付账单 | 状态变 VOID，记录原因和日志 |

| F021 | 退款回调幂等 | 同一退款回调发送两次 | 只生成一条退款流水，账单退款金额不重复累加 |
| F022 | 退款金额拦截 | 退款金额超过可退金额 | 接口拒绝，不生成退款单 |
| F023 | 支付退款对账 | 查询项目对账汇总 | 返回支付金额、退款金额、净入账和异常差额 |
